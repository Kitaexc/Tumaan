Добро пожаловать в игру "Туман"
Перед запуском игры установите пожалуйста библиотеки:
colorama и PyGame
Что бы это сделать, запустить командную строку от имени администратора и впишите в нее команды:
"pip install colorama" и "python3 -m pip install pygame" (без ковычек)
А также вставьте ваш путь к файлу со звуком в 8 строку кода. (Файл со звуком находится в папке)
